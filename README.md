# Breast Cancer Diagnosis with Machine Learning

This project compares supervised and unsupervised machine learning methods for breast cancer diagnosis.

## Models
- Random Forest Classifier
- k-Means Clustering

## Dataset
Breast Cancer Wisconsin Diagnostic Dataset

## Results
Random Forest achieved an accuracy of 96.49%.
k-Means achieved an accuracy of 92.98% after cluster-label mapping.

## Technologies
Python, pandas, NumPy, scikit-learn, matplotlib, seaborn, Jupyter Notebook
